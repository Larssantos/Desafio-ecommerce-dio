# Desafio-ecommerce-dio
Projeto de Banco de Dados - E-commerce
Este é o meu projeto para o desafio da DIO. Eu trabalhei no refinamento de um banco de dados para uma loja virtual.
O que eu fiz de novo: 
Clientes: Diferenciei quem é Pessoa Física e quem é Pessoa Jurídica.
Pagamentos: Agora o cliente pode ter mais de uma forma de pagamento salva.
Entregas: Criei uma tabela para rastrear o pedido com status e código de rastreio.
